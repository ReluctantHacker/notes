# 營養素(nutrient)與營養密度(nutrient density)
常見的人體所需營養素包含, 糖, 脂肪, 蛋白質, 各種維生素等等. 我們定義一種營養素的營養價值, 精神上採用人體所需(每單位時間, 通常以一日為單位)來衡量. 例如健康的人一天需要1g的蛋白質, 需要100g的脂肪, 則假設1g的蛋白質重要性等同100g脂肪, 則1g的大白質其營養價值等於100g脂肪的營養價值, 也就是假設1g的蛋白質其營養價值為1, 則1g的脂肪其營養價值只有0.01. 有時候營養價值會用該營養素的熱量作為分母, 也就是假設健康的人一天需要1-cal熱量的蛋白質, 需要100-cal熱量的脂肪, 來計算相對營養價值. 這也是"垃圾食物"又稱為空熱量食物的由來. 表示他們單位熱量下的營養價值低下.

營養價值的基本精神如上, 但實際目前世界並無統一定義, 但幾種標準精神皆同, 只是細節定義略有出入, 例如Nutrient Profiling, NRF, Nutri-Score等方法. 採用%DV的概念(每日所需占比). 基本與我先前所說概念相同.

 nutrient profiling model (NPM)也是基本相同的概念, 以下表格可以大致看出, 每日所需越少, 營養價值越高

| 營養素    | 每日需求量   | 功能關鍵性    | 單位效能 | 營養價值趨勢（反比觀） |
| ------ | ------- | -------- | ---- | ----------- |
| 維生素B12 | 0.002 g | 造血、神經功能  | 很高   | ⭐⭐⭐⭐⭐       |
| 維生素C   | 0.09 g  | 抗氧化、膠原生成 | 高    | ⭐⭐⭐⭐        |
| 鈣      | 1000 mg | 骨骼、肌肉收縮  | 中等   | ⭐⭐⭐         |
| 蛋白質    | 50 g    | 組織建構     | 一般   | ⭐⭐          |
| 脂肪     | 70 g    | 能量供應     | 低    | ⭐           |
| 糖      | >100 g  | 能量來源     | 很低   | ❌           |

# 溶液
人體中營養和代謝等等機制, 主要就是一系列液體或氣體在溶液中穿透各種不一樣的滲透膜(生物膜)然後發生生物化學反應的總和, 所以我們必須先把溶液的基本概念釐清.

我們常常說A化合物(但一分子在空間呈現重複幾何組合的物體), **溶於**B化合物. 其實是說, 從微觀的分子級別來看, A分子均勻擴散到S空間, 而B分子也均勻擴散整個S空間, 從分子的微觀層級來看, A分子均勻混合B分子. 例如鹽巴溶於水, 糖溶於水. (當然"溶"這個概念不限於液體, 氣體也有一樣的概念)

而所謂非溶解, 按照溶解的定義反推, 則是A物質(可能是某種均勻的化合物), 與B物質混合後, 微觀上並沒有分子層級的均勻分布, 可能某一個區塊都是B, 某一個區塊都是A, 與分子層級的均勻擴散不一樣. 但巨觀上可能看似均勻, 例如牛奶看似均勻, 但微觀上會發現某區塊都是水分子, 而某區塊又都是純牛奶成分. 簡單來說溶質粒子未分散到分子層級, 可能形成微粒, 團塊或懸浮顆粒. 巨觀上, 可能看似均勻, 但實際是異相系統, 例如乳液, 懸浮液, 膠體. 例如牛奶微觀上是脂肪球(大小並不均等)懸浮在水中, 某些區域脂肪密度高, 某些區域幾乎全是水.

若從巨觀下可見溶液是均勻且穩定的(不會很快發生沉澱或明顯不均勻), 我們通常稱之為**膠體(colloid)**, 但膠體根據wikipedia, 並沒有明確定義, IUPAC組織有弄了一個不是很嚴格的定義, 但基本比較像是膠體的一個常見性質整理, 例如粒子大小要是奈米級別, 會有布朗運動等等.  牛奶, 墨水, 蛋白液都是例子.

若是連巨觀下都不能穩定, 則成為懸濁液(Suspension), 但同樣沒有嚴格定義. 例如泥沙, 泥漿, 果汁.

由於膠體是除了純溶液之外最穩定的, 所以她通常也是除了純溶液意外最常見的液體型態.
- 食品: 牛奶、果凍、蛋黃醬、冰淇淋
- 醫學: 血漿、疫苗懸浮液、奈米藥物載體
- 化妝品: 乳液、防曬霜、泡沫潔面
- 空氣: 霧、煙、PM2.5、空氣污染物
- 工業: 墨水、顏料、膠水、氣凝膠

# 固體, 液體, 氣體
經典物理學上, 其定義
- 固體: 微觀下粒子間相對位置不變, 巨觀下形狀與體積不變
- 液體: 微觀下粒子間相對位置可變但每單位體積的粒子數不變(密度不變), 巨觀下形狀可變, 體積不變
- 氣體: 微觀下粒子間先對位置可變且每單位體積的粒子數可變(密度可變), 巨觀下形狀可變, 體積可變

經典力學不考慮分子或原子震動, 因為採用最簡化的模型, 而非經典物理學中(例如量子力學、統計力學、凝態物理等等), 固體, 液體, 氣體之間的定義很難界定, 通常只探討物理現象, 而不嚴格分類相態(沒有意義). 例如凝膠、液晶：同時具有固體和液體特性。超臨界流體：既不像氣體，也不像液體。一般固體也有晶格振動（聲子）、液體有動態結構. 軟體同時具有液體和固體的性質等等.

# 熱量
對人體來說, 所有的熱量從細胞層面來說就是ATP, 而能進入細胞, 且被細胞轉化為ATP的三大人體營養物質, 分別是
- 葡萄糖
- 胺基酸
- 脂肪酸


## 細胞膜構造
細胞膜的運輸[reference](https://bioshchen.weebly.com/320483299033180369393666429289360742604124335.html)
### 磷脂
細胞膜本身主要由兩層磷脂構成, 即是**磷脂雙層**(Phospholipid Bilayer)構成, 每個磷脂分子有兩個部分
- 親水(頭部), 會傾向靠近水分子
- 疏水(尾部), 會傾向遠離水分子

磷脂雙層的構造如下:
```
外層水相
↓
親水頭部
疏水尾部 ←→ 疏水尾部
疏水尾部 ←→ 疏水尾部
親水頭部
↑
內層水相（細胞質）
```
### 蛋白膜
除了磷脂以外, 細胞膜還有蛋白膜, 主要有
- 通道蛋白(channel protein): 開放式通道, 讓特定分子自由通過, 屬於被動選擇
- 載體蛋白(carrier protein): 智能通道, 結合特定分子後改變構型，將其運送到膜另一側, 有選擇性結合，簡單說就是"辨識特定對象後在開門"
- 受體蛋白: 不是通道, 而是按鈕, 告訴細胞要做甚麼事情, 例如特定分子（如激素、神經傳導物質）按下這個按鈕（即結合受體蛋白），細胞就會啟動一連串反應。

它們嵌在在細胞膜上，但不是「完全固定不動」的。
- 細胞膜是流動鑲嵌模型（fluid mosaic model），這是目前最廣泛接受的細胞膜結構模型。
- 根據這個模型，磷脂雙層像液體一樣流動，而膜蛋白（包括載體與通道蛋白）鑲嵌在膜上, 可以在膜面上漂浮移動。

通道蛋白和載體蛋白都屬於運輸蛋白(Transport protein, 也有稱轉運蛋白), 泛指所有幫助物質進出細胞的蛋白(不一定都鑲嵌在細胞膜上).

我們提過的葡萄糖, 胺基酸, 脂肪酸都是透過各自的載體蛋白進入細胞內!
