	A. 行動網路(mobile network)又稱蜂巢式網路(cellular network), 原因便是, 他是基於一座一座基地台(tower)所構成的類似蜂巢狀的結構

	B. 由數個tower連接至一個MSC(mobile switching center), MSC與其MSC之間在互相連接, 形成網路, 他們的連接媒介大部分是用光纖

	C. 行動裝置使用radio(低頻率電磁波), 與tower通信

	D. 使用者附近的MSC會自動偵測行動裝置的位置, 從而讓信號在對應近的tower送出, 到達接收者
	
	E. 目前行動網路已經從1G, 2G, 3G, 4G, 發展到現在的5G了(雖然4G還是佔了多數)

2. Mobile 之間的radio頻譜資源分配(radio很有限的, 要科學的分配)
	A. Frequency slot distribution
		a. 解釋:
			I. 在移动通信中, 频率槽分配是指将可用的频谱资源分配给不同的用户或服务, 这涉及将频谱范围划分为多个频率槽(或频道), 然后将这些槽分配给不同的用户或通信系统, 频率槽分配通常由网络管理者或调度器执行, 以确保各个用户或服务能够在频谱上协调使用

		b. 簡單來說: 就是把有限頻譜謹慎分配給所有tower以及服務
		
	B. Multiple Access Technique
		a. 解釋:
			I. 址接入技术是一种用于在共享的通信介质上允许多个用户同时进行通信的方法。它包括了多个子技术，用于在有限的频谱和时间资源上实现多用户之间的并行通信。常见的多址接入技术包括以下几种：
				🤔. 时分多址(Time Division Multiple Access, TDMA): 将时间分割为若干时隙, 并将每个时隙分配给不同的用户进行通信。

				🤔. 频分多址(Frequency Division Multiple Access, FDMA): 将频谱范围分割为若干频率槽, 并将每个频率槽分配给不同的用户进行通信。

				🤔. 码分多址(Code Division Multiple Access, CDMA): 使用不同的码序列来区分不同用户的数据, 使它们可以同时在相同的频率范围内进行通信。 在CDMA中，不同使用者之間使用相同的頻段和時間進行通信，但通過使用不同的碼（序列）來區分不同的數據流。換句話說，CDMA允許多個使用者在相同的頻段和時間上同時進行通信，而通過使用不同的碼來將它們的數據區分開來。這些碼具有良好的正交性，使得不同的使用者的數據可以有效地區分開來，從而實現同一頻段和時間上的同步通信。以一個直觀的例子來說，可以把CDMA比喻為一群人在同一個大廳裡同時說話，但每個人都使用自己獨特的語言（碼）來表達。其他人雖然聽到了所有的聲音，但只有能夠理解自己語言的人才能理解並回應。這樣就實現了在同一時間和頻段上的多用戶通信。展頻碼在CDMA系統中扮演著識別用戶身份的重要角色。每個用戶都擁有自己唯一的展頻碼，這個展頻碼在傳輸過程中用於將用戶的數據信號進行展頻。
				假設我們有兩個用戶A和B，他們的展頻碼分別是𝐶𝐴CA和𝐶𝐵CB
				展頻碼的生成：展頻碼可以是一個長度為N的二進制序列。例如，我們可以使用如下的展頻碼：
				𝐶𝐴CA=1010110011
				𝐶𝐵CB=1100101100
				信號的展頻：當用戶A要傳送一個位元為1的信號時，他會使用自己的展頻碼
				𝐶𝐴CA將信號進行展頻。展頻的方式是將信號與展頻碼進行運算，通常使用的運算方式是將信號與展頻碼的對應位元進行相乘，然後將結果相加。
				假設A要傳送的信號為1，那麼展頻後的信號就是
				𝑆𝐴SA=1010110011
				同樣地，如果用戶B要傳送一個位元為0的信號，他會使用自己的展頻碼
				𝐶𝐵CB將信號進行展頻，展頻後的信號就是
				𝑆𝐵SB=0000000000。
				通信中的信號：在通信介質上，用戶A和用戶B的展頻信號會同時存在。由於展頻碼的不同，它們的信號在頻域上會呈現不同的特徵。

		b. 簡單來說: 就是把分配到的頻譜進行高效率的利用

3. 所有generation
	A. 第一代(1G):
		a. 模拟信号传输: 1G移动通信系统主要使用模拟信号传输, 通过无线电波在基站和移动设备之间进行语音通信
		b. 频分多址接入(FDMA): 频分多址接入技术用于将频谱划分为不同的频道, 以允许多个用户同时通话
		c. 優缺點:
			I. 优点:
				🤔. 开启了移动通信时代, 是移动通信的先驱。
				🤔. 实现了模拟语音通信, 为后续技术发展奠定了基础。
			II. 缺点:
				🤔. 信号质量不稳定, 通话质量差。
				🤔. 频率资源有限, 容量小, 无法满足大规模用户需求。
			III. 淘汰原因: 
				🤔. 通话质量差, 用户体验不佳。
				🤔. 频率资源有限, 无法满足日益增长的用户和数据需求。

	B. 第二代(2G):
		a. 数字信号传输: 2G引入了数字信号传输, 使用数字编码技术将语音信号转换为数字数据进行传输。
		b. 时分多址接入(TDMA): 时分多址接入技术将时间划分为时隙, 并允许多个用户在不同的时隙上进行通信。
		c. 全球漫游: 2G开始实现全球漫游功能, 使用户能够在不同的国家和地区之间进行通信。
		d. 優缺點:
			I. 优点:
				🤔. 引入了数字信号传输, 通话质量更稳定。
				🤔. 支持全球漫游, 提高了通信的便捷性。
			II. 缺点: 
				🤔. 数据传输速率较低, 无法满足日益增长的数据需求。
				🤔. 频谱利用率低, 容量仍然有限。
			III. 淘汰原因: 
				🤔. 数据传输速率低, 无法满足多媒体和互联网应用的需求。
				🤔. 频谱利用率低, 容量不足以支持大规模用户和数据传输。

	C. 第三代(3G):
		a. 宽带数据传输: 3G引入了宽带数据传输, 支持更高速率的数据通信, 使用户可以享受更快的互联网访问和多媒体服务。
		b. 分组交换: 3G网络采用分组交换技术, 可以更高效地管理网络资源, 支持更多的用户同时连接和通信。
		c. CDMA技术: 3G主要采用CDMA(Code Division Multiple Access)技术来实现多用户间的并行通信, 提高了网络的频谱利用率。
		d. 優缺點:
			I. 优点:
				🤔. 引入了宽带数据传输, 支持更高速的互联网访问和多媒体服务。
				🤔. 支持分组交换和CDMA技术, 提高了网络容量和频谱利用率。
			II. 缺点：
				🤔. 基站建设成本高，覆盖范围有限。
				🤔. 数据传输延迟相对较高，无法满足实时应用需求。
			III. 淘汰原因：
				🤔. 建设成本高，运营商投资回报率低。
				🤔. 无法满足实时性要求高的应用需求。

	D. 第四代(4G):

		a. LTE技术: 4G的主要标准是LTE(Long Term Evolution), 它提供了更高的数据速率, 更低的延迟和更好的网络覆盖, 一下都是LTE所使用到的技術。

		b. OFDMA技术: LTE网络采用OFDMA(Orthogonal Frequency Division Multiple Access)作为其下行链路的多址接入技术, 使得网络可以更有效地分配频谱资源, 支持更多用户同时连接。

		c. SC-FDMA(Single Carrier Frequency Division Multiple Access): 在LTE的上行链路中, 采用了SC-FDMA技术, 相比于传统的多址接入技术, 如CDMA, SC-FDMA具有更低的峰值功率和更高的功率效率, 适合移动设备的电池供电

		d. MIMO(Multiple input mulitiple output): LTE 使用 MIMO 技术来增加数据传输的吞吐量和信号的可靠性。MIMO 允许同时发送多个数据流，并在接收端使用多个天线来接收这些数据流，从而提高了信道容量和数据传输速率。

		e. IP网络: 4G网络采用了全IP(Internet Protocol)架构, 实现了对互联网的无缝接入, 为用户提供了更丰富的互联网应用和服务。(你連撥號打電話, 都是透過IP網路! 也就是所謂的VoLTE,  不用懷疑, 他應該是把手機訊號轉成IP封包, 送到4G基地台, 然後連到電信商的網路, 然後在送到接收端的基地台...我猜. 而一般的電話是透過傳統電話網路, 那個就是2G, 3G的事情了)
`
		f. 優缺點:
			I. 优点：
				🤔. LTE技术实现了更高速率的数据传输，提高了用户体验。
				🤔. OFDMA技术和全IP架构提高了网络容量和频谱利用率。
			II. 缺点：
				🤔. 建设成本高，需要大量基站和频谱资源。
				🤔. 室内覆盖能力较弱，信号穿透性不佳。
			III. 淘汰原因：
				🤔. 室内覆盖能力弱，用户体验不佳。
				🤔. 无法满足日益增长的数据需求和新兴应用的要求。



	E. 第五代(5G): 
		a. 超高速数据传输: 5G网络提供了更高速率的数据传输, 甚至可以达到几十Gbps的速率, 为用户提供了更快的下载和上传速度。
		b. 低延迟通信: 5G网络具有极低的延迟, 可以实现几毫秒甚至亚毫秒级别的通信延迟, 支持实时应用和服务, 如智能交通、远程医疗等。
		c. 大容量连接: 5G网络可以支持大规模的设备连接, 实现物联网(IoT)应用的快速发展, 连接各种智能设备和传感器。
		d. 優缺點:
			I. 优点:
				🤔. 提供了更高速率、更低延迟、更大容量的通信服务。
				🤔. 支持大规模物联网连接和实时应用需求。
			II. 缺点：
				🤔. 基站建设和频谱资源需求更高。
				🤔. 技术标准尚不完善，商用推广面临挑战。
			III. 淘汰原因：
				🤔. 目前尚未淘汰，但可能受到未来更先进技术的挑战。
